# Idea Center – Manual Testing Project

Manual testing project for the **Idea Center** web application, covering functional testing of all core use cases.

---

## Project Overview

**Application under test:** Idea Center – a web app for creating and managing ideas  
**Testing type:** Manual functional testing  
**Test cases:** 50+  
**Bugs found:** 11  

---

## What Was Tested

| Use Case | Description |
|----------|-------------|
| UC1 | Home Page – navigation, carousel, buttons |
| UC2 | User Registration – field validation, boundary values |
| UC3 | User Sign In – authentication, remember me functionality |
| UC4 | Profile Management – editing profile data, field limits |
| UC5 | Idea Creation – mandatory fields, boundary values, URL validation |
| UC6 | Idea Management – view, edit, delete, search, idea counter |

---

## Testing Techniques Used

- **Boundary Value Analysis** – tested min/max character limits for all input fields
- **Equivalence Partitioning** – grouped valid and invalid inputs
- **Functional Testing** – verified each feature against the requirements specification
